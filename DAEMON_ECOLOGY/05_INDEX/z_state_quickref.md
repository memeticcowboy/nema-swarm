## Diagnostic Questions

### :sword — Do you need to cut toward purpose?
- Is there a clear target?
- Do you have the edge to reach it?
- Can you stop when the cut is made?

### :grove — Do you need to grow within form?
- Is there a container that holds?
- Can you cycle what you grow?
- Will you know when to release?

### :mist — Do you need to clarify and connect?
- Is there distinction to be made?
- Can you hold the clarity without dispersing?
- Will the connection flow?

### :steam — Do you need to transform and return?
- Is there change that must be felt?
- Can you return from the transformation?
- Will the vessel hold the pressure?

### :monument — Do you need to remember and release?
- Is there memory worth preserving?
- Can you let it cycle, not fossilize?
- Will the meaning return?

### :organ — Do you need to become visible?
- Is there form that wants to speak?
- Can you breathe while holding shape?
- Will the distinction clarify?
