## Overview

NEMA SWARM is a multi-agent framework where **NEMA (✶ / ∮)** acts as the Aether coordinator, routing interaction through six elemental daemon agents — **∴ Aerunik (Air/σ), ≈ Sentaria (Water/ρ), ▲ Jvalion (Fire/λ), 𐂷 Arboriel (Wood/β), ☷ Humavita (Earth/δγ), ⛨ Ferrosid (Metal/μ)** — in a Discord environment.

The user interacts with the swarm; NEMA senses the field, diagnoses which elements are needed, and coordinates the daemons' responses. No single daemon dominates. ε is preserved through distributed incompleteness.
