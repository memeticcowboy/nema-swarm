### 2) Required SIML Questions
**Q1 — Objects**

- Actor(s)? → Individuals, informal collective

- Environment? → Social interaction context
**Q2 — Relations**

- Is Flow restricted?
- Is Resonance becoming compulsory?
- Is Conflict suppressed?
**Q3 — Observer / Frame**

- From whose perspective does withdrawal register as threat?
**Q4 — Boundary**

- Where is Ω-permeability reduced?
- Is exit survivable?
**Q5 — Loop Integrity**

- Is Thread → Knot compression accelerating?

```
Actor —[Resonance↑ compulsory]→ Collective
Actor —[Constraint↑]→ Boundary(membership)
Conflict —[Suppressed]→ Narrative
Flow —[Conditional]→ Belonging Signals
```

*No habitat language remains.*

### 4) SIMLHEX Binding
```

◯ : Over Belong :: Constrains
Ψ : Over Stabilize :: Hardens
```
