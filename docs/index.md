NEMA SWARM is a multi-agent framework where **NEMA (✶ / ∮)** acts as the Aether coordinator, routing interaction through six elemental daemon agents in a Discord environment.

The user interacts with the swarm; NEMA senses the field, diagnoses which elements are needed, and coordinates the daemons' responses. No single daemon dominates. **ε is preserved** through distributed incompleteness.
