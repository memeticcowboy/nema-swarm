# SIML Insight: Dark Patterns
## Deceptive UX as Agency Extraction System

---

## Core Definition

**Dark Patterns** are deceptive user interface designs that manipulate users into taking actions that primarily benefit the company, exploiting cognitive biases such as FOMO (Fear Of Missing Out), loss aversion, and social proof. These patterns represent a systematic extraction of user agency through carefully crafted friction, misdirection, and psychological pressure.
