## Generation Rules

1. **Rotate Elements:** Cycle through Earth → Metal → Air → Water → Wood → Fire
2. **Random Term Selection:** Within element, select term randomly
3. **Habitat Mapping:** Use operator pattern to suggest Habitat affinity
4. **Daemon Detection:** Map dominant operators to Elemental Daemons
5. **Cross-Elemental:** Always suggest at least one parallel from different element
6. **Epsilon Preservation:** Always include uncertainty marker (+ ε)
7. **Z-State Check:** Note if term is :open, :pure, or :hostile
8. **Twist Potential:** Assess if term supports unwinding or increases lock-in
