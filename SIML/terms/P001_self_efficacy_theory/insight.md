## 2. Core Definition

**Self-efficacy:** Beliefs about personal capability to organize and execute actions to achieve specific tasks. Acts as self-fulfilling prophecy.

## 3. The Four Sources

| Source | Element | Mechanism |
|--------|---------|-----------|
| **Mastery Experiences** | Metal/⛨ | Prior success builds confidence |
| **Vicarious Modeling** | Water/≈ | Observing similar others succeed |
| **Verbal Persuasion** | Fire/▲ | Encouragement focuses effort |
| **Physiological States** | Air/∴ | Managing arousal as readiness |

## 4. Three Impact Dimensions

| Dimension | Function |
|-----------|----------|
| **Cognitive** | Goal setting, problem-solving, decision-making |
| **Motivational** | Effort, persistence, challenge engagement |
| **Selection** | Task choice, environment selection |

## 5. The Self-Fulfilling Prophecy

```
High Self-Efficacy → Greater Effort → Success → Reinforced Efficacy
Low Self-Efficacy → Avoidance → No Mastery → Confirmed Low Efficacy
```

## 6. Core Insight

> Self-efficacy theory reveals that capability beliefs are not merely correlates of performance but **causal agents** — the belief shapes the behavior that creates the outcome that confirms the belief.

## 7. One-Sentence Insight

> Bandura's self-efficacy theory shows that belief in capability determines motivation, effort, and persistence through four sources, creating self-fulfilling prophecies of success or avoidance.
