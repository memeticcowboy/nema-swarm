# Cognitive Load Theory — Core Insight

## The Essential Understanding

Cognitive Load Theory (Sweller, 1988) reveals that **learning is constrained by the severe limitations of working memory**. We cannot simply pour information into learners' heads; we must carefully engineer instructional experiences that respect the architecture of human cognition.

### The Three Loads

1. **Intrinsic Load** (Air/σ): The essential complexity of what must be learned — unavoidable but manageable through sequencing
2. **Extraneous Load** (Air/σ): The noise of poor instructional design — should be eliminated
3. **Germane Load** (Fire/λ + Earth/δγ): The productive effort of schema construction — should be maximized

### Why This Matters

Traditional instruction often ignores cognitive architecture:
- **Text + narration:** Redundant, overloads working memory
- **Split attention:** Forces search between sources
- **Problem-solving for novices:** Wastes resources on trial-and-error

CLT shifts the focus:
- **From:** More content → **To:** Better-structured content
- **From:** Engaging presentation → **To:** Cognitively efficient design
- **From:** Learner effort → **To:** Germane effort only

### The Schema Connection

The ultimate goal: **Automatization through schemas**. When knowledge becomes schema-based:
- Working memory is liberated
- Expert performance becomes possible
- Cognitive resources are available for higher-order thinking

### For Educators and Designers

- **Respect working memory limits** — segment, pre-train, sequence
- **Eliminate extraneous load** — integrate, signal, align
- **Maximize germane processing** — worked examples, guided instruction
- **Adapt to expertise** — what helps novices may hinder experts

### One-Line Takeaway
