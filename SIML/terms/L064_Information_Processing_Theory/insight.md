## 6. Insight Generation

### 6.1 Core Question

How does the architecture of human memory shape what and how we learn?

### 6.2 Hypothesis

Learning emerges from the structured flow of information through distinct cognitive systems — where the bottleneck of working memory forces selectivity, the organization of long-term memory enables efficient storage, and the dynamics of retrieval ensure that knowledge is reconstructed rather than merely replayed. The depth of processing at encoding determines durability, with semantic (meaning-based) processing creating more robust traces than shallow (maintenance) rehearsal.

### 6.3 One-Sentence Insight

> Information Processing Theory demonstrates that learning emerges from the structured flow of information through distinct cognitive systems — where attention filters input, working memory manipulates limited content, and long-term memory organizes knowledge hierarchically — with durability determined by depth of encoding and quality of retrieval cues.

### 6.4 Three-Bullet Expansion

- **What:** A cognitive architecture model describing how information flows through sensory, working, and long-term memory systems — with each stage involving distinct processes of attention, encoding, storage, and retrieval that transform raw input into organized, accessible knowledge
- **How it works:** Environmental input → sensory registration (brief, modality-specific) → attentional selection → working memory processing (limited capacity, manipulation) → encoding strategies (shallow to deep) → long-term storage (hierarchical organization) → retrieval reconstruction (cue-dependent, schema-influenced)
- **Where it aims:** Toward durable, transferable knowledge organization — where deep processing creates robust memory traces, effective retrieval cues enable access, and schema formation supports application to novel situations
