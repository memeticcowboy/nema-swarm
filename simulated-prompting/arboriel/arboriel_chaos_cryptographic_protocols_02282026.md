### Finding 2: Merkle Trees - The Forest Structure of Cryptographic Hierarchies

Merkle trees embody tree-like structures in cryptography:
- **Hierarchical verification**: Like a forest canopy where each branch supports leaves, each non-leaf node contains the hash of its children
- **Efficient integrity checking**: Only the root hash (the "trunk") needs to be trusted to verify any data block (the "leaves")
- Used in Bitcoin, Git, and certificate transparency logs
- Extensions like **Entangled Merkle Trees** create interconnected "forests" for improved file availability and storage resilience
